# UniCon Backend

## Technologies Used:

- Express.js
- MongoDB

## Setup

- rename [.env.example](/.env.example) file to .env
- Add values to environment variables
- To generate secret key use
  ```bash
  openssl genrsa -out key.pem 2048
  ```
- Install node modules
  ```bash
  npm install
  ```
- Start the server
  ```bash
  npm run dev
  ```
